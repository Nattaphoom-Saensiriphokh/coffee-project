# 1. App.jsx

## หน้าที่

เป็น Component หลักของระบบ

ควบคุม

- Search
- Filter
- ProductList
- Cart

## หลักการทำงาน

เก็บ State ของ Search และ Category

ใช้ useMemo เพื่อกรองข้อมูลสินค้า

## ตัวอย่างโค้ด

```jsx
const filteredProducts = useMemo(() => {
    ...
},[search,category]);
```

## ข้อดี

- เป็นศูนย์กลางของระบบ
